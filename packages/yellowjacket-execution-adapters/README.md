# yellowjacket-execution-adapters

Phase 1 package skeleton for YellowJacket-side execution provider integration.

## Current scope
This package currently defines:
- adapter interfaces
- provider-neutral contracts
- Hermes-local placeholder modules
- package layout for later implementation

## Not yet implemented
- live Hermes HTTP transport
- request/response mapping logic
- health polling behavior
- cancellation behavior
- result normalization behavior
